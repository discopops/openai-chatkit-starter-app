# Domain allowlist (OpenAI) quick reference

Use when ChatKit UI shows domain verification errors.

Key points:
- Add UI domain(s) to OpenAI allowlist.
- Use the domain key (domain_pk_...) in VITE_CHATKIT_API_DOMAIN_KEY.
- Redeploy UI after updating env vars.

Source:
```
https://platform.openai.com/settings/organization/security/domain-allowlist
```
