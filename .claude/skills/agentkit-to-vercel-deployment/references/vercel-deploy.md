# Vercel deployment quick reference

Use when deploying from CLI or UI.

Key points:
- `vercel` or `vercel deploy` performs a deployment.
- `--prod` creates a production deployment.
- Projects can be linked with `vercel link` or created via CLI/UI.
- Build logs are in the Vercel dashboard deployment view.

Sources:
```
https://vercel.com/docs/cli/deploy
https://vercel.com/docs/deployments/overview
```
