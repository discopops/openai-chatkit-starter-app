# ChatKit widgets (summary)

Use this for widget concepts, actions, and server handling.

Key points:
- Widgets are JSON UI blocks (cards, lists, text, forms).
- Use Widget Builder to prototype and export JSON.
- Widgets can trigger actions via ActionConfig.
- Handle widget actions on the server via ChatKitServer.action.
- Upload assets via your backend; hosted uploads supported.

Source:
```
https://platform.openai.com/docs/guides/chatkit-widgets
```
