# AgentKit overview (OpenAI)

Use this for high-level positioning and links to the core docs.

Key points:
- AgentKit is the umbrella for Agent Builder, ChatKit, and Agents SDK.
- Agent Builder is the visual workflow designer.
- ChatKit embeds agentic chat in your product UI.
- Agents SDK is for custom, code-first agents and orchestration.

When to read this file:
- Before deciding whether to use managed workflows vs self-hosted.
- When explaining the overall stack to teammates.
