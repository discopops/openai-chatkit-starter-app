# ChatKit actions (summary)

Use this for action handling patterns.

Key points:
- Actions can be triggered by widget interactions or sendAction.
- Server-side handling via ChatKitServer.action.
- Client-side actions require handler="client" in ActionConfig.
- Form widgets pass values into action.payload by field name.
- Loading behavior can be self, container, none, or auto.

Source:
```
https://platform.openai.com/docs/guides/chatkit-actions
```
