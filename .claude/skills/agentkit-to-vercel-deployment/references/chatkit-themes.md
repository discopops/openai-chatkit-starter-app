# ChatKit theming and customization

Use this for UI theming options and UX tweaks.

Key options:
- theme: colorScheme, accent colors, radius, density, typography.
- start screen greeting and prompts.
- composer placeholder and attachments config.
- custom header buttons.
- toggle history/header regions.
- locale override.

Note: hosted attachments required for the recommended integration.

Source:
```
https://platform.openai.com/docs/guides/chatkit-themes
```
